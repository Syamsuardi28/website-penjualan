âš ï¸ TROUBLESHOOTING:
-------------------

MASALAH: Produk tidak muncul
SOLUSI: 
- Pastikan app.js ter-load (cek di Network tab F12)
- Cek console untuk error
- Pastikan path js/app.js benar

MASALAH: CSS tidak apply
SOLUSI:
- Pastikan styles.css ada di folder css/
- Cek path di HTML: <link rel="stylesheet" href="css/styles.css">
- Clear cache browser (Ctrl+F5)

MASALAH: Cart tidak tersimpan
SOLUSI:
- Pastikan localStorage tidak diblokir
- Jangan gunakan mode private/incognito untuk test persistensi
- Clear localStorage: localStorage.clear() di console

MASALAH: Gambar tidak muncul
SOLUSI:
- Pastikan koneksi internet aktif (gambar dari Unsplash CDN)
- Tunggu beberapa saat untuk loading
- Cek Network tab di DevTools
